# Pesquisa de Satisfação - Empresa TudoWeb

Este projeto foi desenvolvido como atividade prática da Agenda 8 da disciplina de Desenvolvimento de Sistemas da **ETEC**.

## 📝 Descrição do Projeto
O objetivo é coletar dados de uma pesquisa de opinião sobre o atendimento de uma empresa de marketing. O programa utiliza estruturas de repetição e decisão para processar a satisfação dos clientes.

O programa solicita:
- Nome do entrevistado
- Idade
- Opinião (1: Excelente, 2: Bom, 3: Ruim)

Ao final, exibe a quantidade total de respostas "Excelente" e "Ruim".
Observação sobre a Opção 2 (BOM): Conforme solicitado no enunciado, a opção "BOM" é aceita pelo sistema para fins de coleta, mas não interfere nos contadores finais exibidos na tela.

## 🚀 Tecnologias Utilizadas
- ![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
- Estrutura de repetição `for`
- Estruturas condicionais `if`, `elif`

## ⚙️ Como Funciona
O código está configurado para processar 50 entrevistados, conforme a regra de negócio da empresa TudoWeb. 
*(Nota: Durante o desenvolvimento, foram realizados testes com 10 entrevistados para validar a lógica dos contadores, conforme exigido pelo professor).*
